## Added

- First-run onboarding in CLI (one-time welcome box with provider detection) and menu bar (welcome section + Configure Paths action).
- Provider remediation model (`remediation`) with CLI hints and menu `⚠ Fix` actions.
- DMG packaging script with optional signing/notarization support.
- Homebrew tap assets (Formula + Cask) and publication helper scripts.
- Public contribution docs and GitHub issue templates.
- README visual evidence assets (screenshots + animated pressure transition GIF).

## Changed

- `install.sh` now supports explicit signing controls (`--sign-identity`, `--notarize-profile`) and `--unsigned` fallback.
- Runtime state now tracks onboarding lifecycle timestamps.
- Snapshot formatting now surfaces provider errors and remediation steps.

## Fixed

- Menu refresh UI flow remains responsive while onboarding and fix actions are active.
- Missing/inaccessible provider paths now emit direct path-specific guidance.
- Source issue handling no longer degrades into generic "unavailable" without recovery steps.
