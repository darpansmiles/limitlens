## Added

- Native Swift CLI executable (`limitlens`) with human, JSON, and watch modes.
- Native macOS menu bar executable (`LimitLensMenuBar`) with threshold alerts and launch-at-login support.
- Core and menu support test runners.

## Changed

- Unified core architecture for provider adapters, snapshot formatting, and threshold semantics.
- Installation flow now builds universal binaries for Apple Silicon and Intel.

## Fixed

- Notification delivery policy now reflects macOS authorization state.
- Launch-at-login behavior now reports actionable failure details.
- External provider command loading is now explicit-opt-in and validated.
